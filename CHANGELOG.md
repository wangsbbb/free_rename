# Changelog

## 1.0.8

- 预览后台线程新增进度回传，状态栏和进度条会显示当前预览推进情况
- 历史页新增搜索、导出、打开日志位置与清空功能
- 记住当前页面、规则页签、主分栏状态与预览表头宽度
- 大列表时减少列宽自动测量带来的刷新开销
- 启动时按尾部读取最近历史，日志变大后启动更稳

## 1.0.7

- 新增统一的“取消任务”按钮，支持取消预览、扫描与执行任务
- 执行取消时增强文件状态回退，严格模式整体回滚，容错模式回滚未写入项
- 历史记录自动落盘，软件重启后仍保留最近操作
- 关闭窗口时提示先取消后台任务，避免误关
- 程序与文档版本统一更新到 1.0.7

## 1.0.6

- 预览表格切换为 `QTableView + QAbstractTableModel`
- 文件夹扫描改为后台线程执行
- 执行阶段直接复用当前预览结果，不再同步重算规则
- 新增 `workers.py`，进一步分离 Worker 与文件管理逻辑
- 打包脚本与 spec 增加 `styles/` 资源

## 1.0.5

- 正式拆分为 `rule_engine.py`、`file_manager.py`、`ui_main.py`
- `遇错继续` 进入单文件容错模式
- 文件操作进一步下沉到 `FileManager`

## 1.0

- 整理为可直接上传 GitHub 的源码结构
- 统一版本号为 1.0
- 清理本机绝对路径与构建日志残留
- 重写 `free_rename.spec` 为相对路径
- 新增 `.gitignore`、`README.md`、`version_info.txt`
- 保留图标资源与 Windows 运行 / 打包脚本
